Order Fulfillment Distribution DataMart (DDM)
input

1. DDM data is NOT Available for legal purposes. Contact the source system business owners for data required for GM Legal. Go to for a list of business owners.

2. DDM does not retain query results. Requesters must retain all reports in accordance with ILM. However, criteria selected for each query is in the history tab.

3. Requesters must handle their own Excel formatting of the data provided

4. Only options that are listed on Dealer Order Acknowledgements are available in DDM. DDM recieves the options ordered initially by the dealer (event 1000/1100) with a refresh of options when the order is produced. (event 3800)

5. Requesters must update ongoing report criteria as changes are needed. Examples include new model year, new allocation groups, new options etc. These updates are not automatic.

6. Notify the DDM team to cancel an ongoing or routine report that is not longer required.
